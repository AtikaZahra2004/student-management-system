# 🎓 Student Management System

## 📌 Description
This is a full-stack Student Management System built using Django and React.  
It allows users to manage student data efficiently with features like add, update, delete, and view student records.

---

## 🚀 Features
- 🔐 User Authentication (Login/Signup)
- ➕ Add Student Details
- ✏️ Update Student Information
- ❌ Delete Student Records
- 🔍 Search Students
- 📊 Dashboard View

---

## 🛠️ Tech Stack
### Backend:
- Python
- Django
- Django REST Framework

### Frontend:
- React
- HTML, CSS

### Database:
- SQLite / MySQL
